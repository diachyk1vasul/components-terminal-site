Опис

Cash Code MFL – для прийому купюр різної ширини.

Використовується універсальна валідаторна головка MFLV із механізмом, що центрує банкноту. Ширина каналу автоматично регулюється для кожної банкноти, що приймається. Така головка використовується в тому випадку, якщо ширина банкнот різних номіналів неоднакова. Для банкнот завширшки 62-82 мм.
Купюроприймачі Cash Code FL та MFL встановлюються на задній або бічній стінці обладнання. Слова "Front Load" означає, що в цих купюроприймачах стекер (касета для укладання грошових купюр) виймається з тієї ж сторони, що і вставляється купюра.

Особливості купюроприймача Cashcode MFL: - самонастроююча ширина купюроприймального каналу; - валідаторна головка автоматично налаштовується на кілька видів валют; - Система автодіагностики; - високошвидкісний прийом купюр із будь-якої сторони подачі; - Противандальний захист. 
Технічна характеристика купюроприймача MFL:

Ширина приймального каналу 67 or 71mm wide (fixed-width)
Напрямки прийому купюр Усі 4 приймає купюри під кутом
Рейтинг прийому 96% or higher
Час прийому купюри – 1,1 сек.
Депонент (Escrow) One bill
Пам'ять Memory Stick
Інтерфейси Імпульсний, Паралельний, MDB, CCNET, ID003, Single Price
Налаштування ПК та КПК
Робоча напруга 1.2 А
Тип датчиків ємнісні, зчитує штрих-код, УФ
Кешбокс (стекер) 600/1000/1500 із замками
Підключення 18-контактний роз'єм та 6-контактний роз'єм
Блок живлення DC12В (+/-1В)
Драйвери TEST CCNET
Габарити (мм) 104 x 266 x 87mm
Вага 1,2 кг (1000) 1,8 кг (1500)
Температурний режим 3°C~50°C
